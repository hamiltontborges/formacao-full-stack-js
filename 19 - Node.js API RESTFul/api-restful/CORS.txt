CROSS-ORIGIN RESOURCE SHARING (CORS)

Google: "é um mecanismo que permite que recursos restritos em uma página da web sejam recuperados por outro domínio fora do domínio ao qual pertence o recurso que será recuperado".

API rodando em http://www.nossaapi.com.br/api

Front-end rodando em http://www.nossaaplicacao.com.br

Se API for pública, suporta qualquer endereço. Mas se a API for privada, ela deve permitir o domínio que está requisitando.
